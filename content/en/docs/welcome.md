This is a dummy documentation page to verify that Docsy renders correctly on GitHub Pages.

## What is Docsy?

Docsy is a Hugo theme designed for technical documentation. It provides:

- A responsive layout with sidebar navigation
- Built-in search
- Multi-language support
- Dark mode toggle

## Local development

```bash
npm install
npm run serve
```

Then open [http://localhost:1313](http://localhost:1313).

## Deployment

Push to the `main` branch and GitHub Actions will build and deploy the site to `https://maods2.github.io/docsy-test/`.

Make sure GitHub Pages is configured to deploy from the **gh-pages** branch.
